# LimeSpider
Food blog website
Limespider is a custom-built food blog website that combines clean design, responsive architecture, and a passion for storytelling through cuisine. 
From vibrant recipe pages to engaging food narratives, Limespider creates a flavorful experience for every user. 
Developed using modern web technologies, it's both visually appealing and performance-optimized.
Built with HTML, CSS and JavaScript, this project reflects my commitment to intuitive user experiences and expressive front-end development.
Whether you’re a seasoned chef or a curious beginner, Limespider invites you to explore, savor, and share your culinary journey.
